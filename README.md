# Platformowka

A 2D platformer GLOOP game developed in Unity as a personal project. The game features multiple levels, character movement, jumping mechanics, animations, and a level selection system.

## Features

- Character movement
- Jumping mechanics
- Animated player character
- Tilemap-based level design
- Multiple playable levels
- Level selection menu
- Win screen after completing the game
- Scene management system

## Technologies

- Unity
- C#
- TextMeshPro
- Unity Tilemap System
- Unity Input System
- Git

## How to Run

### Open in Unity

1. Clone the repository:

```bash
git clone https://github.com/Tssara997/UnityPlatformer.git
```

2. Open the project using Unity Hub.

3. Open the main menu scene.

4. Press Play in the Unity Editor.

### Build

To create a standalone build:

1. Open Unity.
2. Go to:

```text
File → Build Profiles
```

3. Select your target platform.
4. Click **Build**.

## Controls

| Key | Action |
|------|---------|
| A / D | Move |
| Space | Jump |
| R | Die and set a platform |
